

## 1️⃣ var vs let vs const
- var → function-scoped, redeclare & reassign allowed  
- let → block-scoped, reassign allowed, no redeclare  
- const → block-scoped, no reassign, must init  

## 2️⃣ Spread Operator (...)
- Expand/copy/merge arrays or objects

## 3️⃣ map(), filter(), forEach()
- map() → returns new array after transforming  
- filter() → returns new array based on condition  
- forEach() → iterates, no return  

## 4️⃣ Arrow Function
- Shorter function syntax, no own `this`  

## 5️⃣ Template Literals
- Backticks (`) + ${} for variables and multiline strings
